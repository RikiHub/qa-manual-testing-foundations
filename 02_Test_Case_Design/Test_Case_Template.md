| Campo (Descripción) 
|------- ()
| Test Case ID (Identificador único del caso de prueba)
| Test Case Name (Nombre claro y descriptivo del caso)

| Requerimiento  (ID o descripción del requerimiento)

| Precondiciones  (Estado previo necesario para ejecutar la prueba)

| Autor  (QA Analyst)
