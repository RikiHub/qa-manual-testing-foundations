# QA Manual Testing & Strategy Portfolio

Este repositorio forma parte de mi portafolio como QA Analyst y está enfocado
en pruebas manuales, diseño de casos de prueba y estrategia de testing.

## ¿Qué encontrarás aquí?
- Diseño de planes y estrategias de prueba
- Casos de prueba manuales bien estructurados
- Análisis de riesgos
- Reportes de ejecución y defectos
- Ejercicios de testing exploratorio
- Simulación de proyectos reales

## Objetivo
Demostrar mi capacidad para analizar requerimientos, definir una estrategia
de testing efectiva y asegurar la calidad del software desde una perspectiva
funcional y de negocio.
